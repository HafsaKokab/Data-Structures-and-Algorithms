This folder contains my C++ practice of the Standard Template Library (STL).

## Topics Covered

- Pair
- List
- Set
- Multiset
- Unordered Multiset
- Map
- Multimap
- Unordered Map

## Summary

| Container | Description |
|-----------|-------------|
| **set** | Unique elements, Sorted |
| **multiset** | Duplicate elements allowed, Sorted |
| **unordered_set** | Unique elements, Unsorted (Hashing) |
| **unordered_multiset** | Duplicate elements allowed, Unsorted (Hashing) |
| **map** | Unique keys (Key-Value Pair), Sorted |
| **multimap** | Duplicate keys allowed, Sorted |
| **unordered_map** | Unique keys (Key-Value Pair), Unsorted (Hashing) |
