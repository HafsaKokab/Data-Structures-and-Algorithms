# Heap

This folder contains my C++ implementation of the Heap Data Structure.

## Topics Covered

- Max Heap
- Min Heap
- Insert Operation
- Delete Operation
- Heapify

## Summary

### Max Heap
- Complete Binary Tree (CBT)
- Parent node is greater than or equal to its child nodes.
- The largest element is always at the root.

### Min Heap
- Complete Binary Tree (CBT)
- Parent node is less than or equal to its child nodes.
- The smallest element is always at the root.

## Time Complexity

| Operation | Time Complexity |
|-----------|-----------------|
| Insert | O(log n) |
| Delete | O(log n) |
| Heapify | O(log n) |
| Peek (Root) | O(1) |
| Search | O(n) |
